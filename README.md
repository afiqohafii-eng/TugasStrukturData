# Tugas Personal 2 - Struktur Data

##  Deskripsi

Program ini dibuat untuk memenuhi tugas mata kuliah Struktur Data dengan topik:

* Linked List
* Stack dan Queue

Bahasa yang digunakan adalah Java.

---

##  1. Program Linked List (Data Buku)

Fitur:

* Tambah buku
* Hapus buku terakhir
* Cari buku berdasarkan kode
* Tampilkan semua buku

Struktur:

* Node.java
* LinkedList.java
* MainLinkedList.java

---

##  2. Program Stack & Queue (Sistem Kasir)

Fitur:

* Tambah antrian pelanggan (Queue)
* Layani pelanggan (Dequeue)
* Simpan riwayat transaksi (Stack)
* Tampilkan data

Struktur:

* Node.java
* Queue.java
* Stack.java
* MainKasir.java

---

## ▶ Cara Menjalankan Program

### Linked List

cd linkedlist
javac *.java
java MainLinkedList

### Kasir

cd kasir
javac *.java
java MainKasir

---

##  Catatan

Program dibuat dengan konsep sederhana sesuai materi perkuliahan dan sudah diuji berjalan tanpa error.
